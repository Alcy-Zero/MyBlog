# hugoblog_by_hugo
这里是一个hugo博客，想要安装在你的机器上，你必须先安装docker，或者dockertoolbox。

$ git clone git@github.com:Alcy-Zero/MyBlog.git
			
$ cd MyBlog
  
$ sh bilud.sh

$ sh run.sh

$ sh stop.sh

```
**访问**
dockertoolbox用户可通过docker-machine的IP地址加1313端口访问：

例如：
http://192.168.99.100:1313/

douker用户可直接通过：

http://localhost:1313/
